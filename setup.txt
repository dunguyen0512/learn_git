########### git comments  ##############

git add  - staging your changes. 
1/ chose the file to storage into git history (staging area). 
2/ Move files from working area -> staging area
3/ create an album before "commit"

git add "filename" :add single file
git add . : add all changes file
git add src/ : add a source 

git commit -m "message to operator" -> move file from staging area to local repository with a message

############ logging#########################
checking git history includes git commits and important changes.

git log # check history
git log --oneline # shows summary 
git log -Number # show Number latest commits.
git log --author="user.name" # check author's commits.

#### git push #####

first time with "git push -u origin main" #-u set upstream, established tracking, origin is the remote name. branch is main.
second time, use "git push".
